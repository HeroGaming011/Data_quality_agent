# AI Data Quality Agent - Master Prompt

## Project Objective

Build an AI-powered Data Quality Agent that allows users to upload CSV
datasets, automatically detect data quality issues, generate intelligent
recommendations using an LLM, and produce Python data-cleaning scripts.

## Core Features

1.  CSV file upload.
2.  Dataset preview.
3.  Automatic schema detection.
4.  Missing value detection.
5.  Duplicate record detection.
6.  Mixed datatype detection.
7.  Invalid email format detection.
8.  Outlier detection using IQR.
9.  Age validation (0--100).
10. AI-generated recommendations.
11. AI-generated Python cleaning scripts.
12. Dashboard-style UI using Streamlit.

## Technology Stack

-   Python
-   Pandas
-   Streamlit
-   OpenAI/OpenRouter LLM
-   HTML/CSS Styling
-   GitHub

## Functional Requirements

When a user uploads a CSV: - Read the dataset. - Detect data quality
issues. - Display a dataset preview. - Show column metadata. - Generate
issue reports. - Send issue summaries to the LLM. - Return
recommendations. - Generate executable Pandas cleaning code.

## Expected Output

-   Dataset Preview
-   Quality Report
-   Column Summary
-   AI Recommendations
-   Generated Python Script

## GitHub README Description

AI Data Quality Agent is an intelligent data validation platform that
combines rule-based quality checks with Generative AI to identify
anomalies, provide recommendations, and automatically generate
data-cleaning scripts.
